Features

あなたのLP作りを、
まるごと助ける

機能ではなく、あなたが得られる変化を紹介します。

🎨

そのブランドの色がそのまま出る

対象サイトを読み取って、カラーパレットを自動で抽出。ブランドの世界観をそのまま再現する。

✍️

文章をゼロから考えなくていい

サイトのテキストを読み取り、AIがLP向けの文章に書き直す。コピーライターいらず。

🚀

ボタン1つでそのまま公開

GitHub Pagesへワンクリック公開。サーバー設定もアップロードもいらない。

🎬

YouTube動画にも対応

YouTubeのURLを貼れば、字幕を読み取って動画紹介ページを作る。

🔗

複数URL・画像・PDFもOK

1つのサイトだけでなく、複数のURLや画像、PDF、Google Mapsからも情報を集められる。

📝

Notionに記録が残る

生成したLPの情報をNotionに自動で保存。あとから見返すのもかんたん。

How it works

3ステップで完成

難しい操作は一切なし。

01

URLを貼る

対象サイトのURLを入力欄に貼り付ける。それだけ。

02

AIが読み取って作る

色・フォント・文章を自動で抽出して、LPを組み立てる。

03

ボタンで公開

GitHub Pagesへワンクリックで公開。すぐにURLが手に入る。
